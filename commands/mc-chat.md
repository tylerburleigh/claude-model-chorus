# ModelChorus Chat

Quick access to multi-model conversational AI for brainstorming, technical discussions, and getting second opinions.

## Usage

```
/mc-chat [your question or topic]
```

## What It Does

Launches the ModelChorus chat skill to:
- Get second opinions from external AI models
- Brainstorm solutions and ideas
- Discuss technical approaches
- Validate designs and decisions
- Explore complex concepts

## Examples

```
/mc-chat What's the best caching strategy for a high-traffic API?
```

```
/mc-chat Review my authentication approach: JWT tokens with Redis sessions
```

```
/mc-chat Explain the trade-offs between microservices and monolithic architecture
```

## Features

- Choose from multiple AI models (Gemini, GPT-5, Codex, etc.)
- Multi-turn conversations with context
- Adjust thinking depth and creativity
- Share files for code review
- Get detailed, thoughtful responses
